## LearnHub - Landing Page

LearnHub is a modern and responsive landing page built using **HTML5** and **CSS3**. It is designed for an online learning platform and showcases a clean layout with a hero section, features, about section, testimonials, and a footer.

## Features

* Sticky navigation bar
* Hero section with headline and Call-to-Action button
* Features section
* About section
* Modern UI with a consistent color scheme
* Clean and well-structured code

##  Technologies Used

* HTML5
* CSS3


##  How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Explore the landing page.

##  Sections Included

* Home
